## 動作環境
動作に必要なミドルウェアは下記の通りです。

|名称|バージョン|
|:---|:---|
|Python|3.4.2|
|bottle|0.12.11|
|Redis server|2.8.17

## ディレクトリ構成
リポジトリ内は以下のディレクトリで構成されます。

|パッケージ名|説明|
|:---|:---|
|doc|ドキュメントを格納|
|common|共通パッケージ|
|dao\interface|DBアクセスインターフェースパッケージ|
|dao\implement|DAO実装クラスパッケージ|
|dto|データ転送オブジェクトパッケージ|
|function|bottleで使用するファンクションパッケージ|
|template|bottleで使用するHTMLテンプレート|
